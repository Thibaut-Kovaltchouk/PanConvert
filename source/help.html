Panconvert: Helpfile

Supported Formats:

All by Pandoc supported formats can be used. See: UserGuide Pandoc

To use all not explicitly listed formats of pandoc, you have to use the manual converter.

Usage of the manual converter







































  1. The Standard – Checkbox has to be unchecked.

  2. In the from field you have to fill in the supported pandoc formats of the source format.

  3. In the to field you have to fill in the supported pandoc formats of the destination format

  4. For some formats the Parameter field can be left blank, for others it has to be filled. To get a odt file, you have to specify the path, where the file will be saved and the name of the file. There is a working example filled in already. The file will be saved in the program working directory.

Path to pandoc

The program tries to detect where your pandoc executable is installed. If that fails, you can use the preferences dialog in the Edit – Menu to manually save the path to the executable.

Platforms tested:

  1. Windows 7 (32bit)

  2. Mac OS: Lion (32bit), Mountain Lion (64bit), Mavericks (64bit)

  3. Linux Mint 17: Ubuntu

Not Running: (Linux Mint Debian: does not work without tweaking the ui-code of the main ui-file. It is a problem of the installed QT-Version)

System Requirements

These additional Software-Packages have to be installed for running PanConvert.

  1. Python3

  2. QT5

  3. PyQT5

  4. Pandoc

Some older Versions of Python3 or QT5 might not work
